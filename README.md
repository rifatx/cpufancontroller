# CPU Fan Controller Bash Script

Bash script that turns a cpu cooling fan (or any other device connected to a transistor driven circuit) through a GPIO pin. Originally written for Orange Pi 4 LTS, but can be adapted to any device that runs \*nix and has gpio pins. 

## Circuit Diagram

![circuit](circuit.png)

## Installation

Run `sudo install.sh`

Script will be installed as a system daemon.
